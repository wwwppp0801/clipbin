# v0.2.0 Feature Summary

**Date**: 2026-03-29

## New Features Since v0.1.0

### Content Types
- **Rich Text (HTML)** — Web page copies detected via NSPasteboard `public.html`
- **File URLs** — Finder file copies read via NSPasteboard NSURL API
- **Image PNG** — Images encoded via `image` crate, stored as BLOB

### Clipboard Management
- **Pin/Unpin** — Right-click → Pin. Pinned clips sort first, survive clear/limit
- **Max clips limit** — Enforced from settings (default 500), deletes oldest unpinned
- **Clear All** — Footer button, preserves pinned clips

### Keyboard
- **1-9 quick paste** — Number badges on cards, press to paste instantly
- **Cmd+C** — Copy selected clip to clipboard without pasting or closing
- **Delete/Backspace** — Remove selected clip
- **Escape** — Dismiss panel
- **Arrow keys** — Navigate between cards
- **Enter** — Paste selected

### UI Polish
- **Footer bar** — Shows clip count, pinned count, Clear All button
- **File cards** — Show file icon + filename (not full path)
- **Rich text** — Green badge, sans-serif font
- **Empty state** — Clipboard icon + contextual message + hotkey hint
- **Toast notification** — "Copied to clipboard" on Cmd+C
- **Settings** — Keyboard shortcuts cheat sheet, version display
- **Number badges** — First 9 cards show shortcut number

### Paste Implementation
- Write to clipboard via arboard (text/image) or NSPasteboard (file URLs)
- Remember frontmost app PID on panel open
- Activate previous app via NSRunningApplication
- Simulate Cmd+V via CGEvent (like Maccy)
- Self-triggered flag prevents re-recording

### Window
- Position above Dock via NSScreen.visibleFrame
- Auto-hide on blur with 400ms grace period
- Slide-up/down animations via Tauri events
- Template tray icon (44px @2x, icon_as_template)

### CI/CD
- Lint (clippy + eslint + prettier + tsc) → Test (cargo + vitest) → Build
- All passing on GitHub Actions
